# GoPro-API
Library for GoPro camera communication.

# Functions
- Connection initialistaion (searching for GoPro bluetooth address, turning on WiFi and connecting to this network)
- Power off
- Start recording
- Stop recording
- Battery status
- Busy status
- Recording status
- Remaining space
- System ready status

# Compatibility
This library was tested with:
- GoPro Hero 7
- GoPro Hero 8
